# Proyecto: Adivina el Número (NÚMERO MÁGICO) 🐍

Este es un programa en *Python* que utiliza lógica de búsqueda binaria para adivinar un número entre 1 y 100 en el menor número de intentos posible.

### ✨ Lo más destacado:
* *Lógica Inteligente:* El algoritmo reduce las opciones a la mitad en cada paso.
* *Control de Errores:* Detecta si el usuario da pistas contradictorias.
* *Arquitectura:* Diseñado previamente con diagramas de flujo para un código limpio.

### 🛠️ Herramientas:
* *Lenguaje:* Python 3.x
* *Entorno:* Spyder 
* *Diseño:* Diagrama de Flujo
